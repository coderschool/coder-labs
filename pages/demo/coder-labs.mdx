## Code Reference

Sometimes it is useful to reference code from a section of text. We need to wrap the text and the code inside an element: `CH.Section`

<CH.Section>

```python
primes = []
for possiblePrime in range(2, 100):
  isPrime = True
  for num in range(2, possiblePrime):
    if possiblePrime % num == 0:
      isPrime = False
  if isPrime:
    primes.append(possiblePrime)
```

- First we [initialize a list](focus://1).
- Then inside a loop from 2 to 100 we [assume the number is prime](focus://3),
- Until we can [prove it is not](focus://4:6).
- If we can not then we [add the number to the list](focus://7:8)

</CH.Section>

### Intro to flexbox

Let's create 4 boxes with the HTML `div` element.

```html index.html
<div class="container">
  <div class="box box-one">1</div>
  <div class="box box-two">2</div>
  <div class="box box-three">3</div>
  <div class="box box-four">4</div>
</div>
```

---

Then we work on the CSS code. First of all we apply the rule

_`box-sizing: border-box`_

to all elements in our website.

```html index.html focus=2:10
<style>
  :root {
    box-sizing: border-box;
  }

  *,
  ::before,
  ::after {
    box-sizing: inherit;
  }
</style>
<div class="container">
  <div class="box box-one">1</div>
  <div class="box box-two">2</div>
  <div class="box box-three">3</div>

```

  // Declare a new state variable, which we'll call "count"
  // mark
  const [count, setCount] = useState(0);
  return (
    <div className="App">
      <h1>Counter</h1>

      <button onClick={() => setCount(count + 1)}>Click me</button>

  );
}
