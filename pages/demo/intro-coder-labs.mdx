## Getting started

**Prerequisites:**

- <a href="https://nodejs.org/en/" target="_blank">NodeJS</a>
- git
- Access to <a href="https://github.com/coderschool/coder-labs" target="_blank">the Github repo</a>
- Understanding basic <a href="https://reactjs.org/" target="_blank">ReactJS</a>
- Understanding basic <a href="https://mdxjs.com/" target="_blank">MDX</a>

**Clone the project:**

```bash focus=1
git clone https://github.com/coderschool/coder-labs.git
```

<CH.Section>

**Start the project:**

```bash
cd coder-labs
npm install
npm run dev
```

- [Move to the project folder](focus://1).
- [Install libraries and dependencies](focus://2), you only need to run this command once.
- [Start the development server](focus://3).

</CH.Section>

**Open <a href="http://localhost:3000/demo/coder-labs" target="_blank">localhost:3000</a>**
